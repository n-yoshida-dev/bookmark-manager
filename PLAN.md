# PLAN — ブックマーク管理アプリ

## 何を作るか
URLとメモを保存・一覧・検索・削除できるシンプルなブックマーク管理アプリ。

## なぜ作るか
- Claude Code でのバイブコーディング体験 → 公開（リリース）までを一通り経験するため
- 学習中の Go（バックエンド）と React（フロントエンド）をフルスタックで実地に使うため

## スコープ（最初の1本）
- 認証なし（自分専用の単一ユーザー想定）
- 機能は4つだけ：追加 / 一覧 / 削除 / 検索
- DB は SQLite（ファイル1個、サーバー構築不要）

## やらないこと（あえて切る）
- ユーザー認証・ログイン
- タグ・フォルダ分け
- OGP取得・サムネイル
